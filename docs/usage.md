# Usage

## Install

```bash
pdm install -G dev
```

## Run tests

```bash
pdm run test
```

## Run the MCP server

```bash
pdm run python -m polymarket_mcp.server
```

## Run docs locally

```bash
pdm install -G docs
pdm run docs
```

## FastMCP CLI

```bash
fastmcp run src/polymarket_mcp/server.py:mcp
```
